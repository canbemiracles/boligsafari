Multiple Attachments Email

Send or Receive HTML Email with Attachments

  • Created: 03/17/2013
  • Latest Update: 06/26/2014
  • By: Techmynd
  • Email: admin@techmynd.com

HTML5 PHP Multiple Attachment Email lets you send fully supported HTML email with multiple attachments. Get this script, customize settings and upload at your server or integrate it in your current project to send or receive HTML based emails containing multiple file attachments. GIF, PNG, JPG, RAR, ZIP, PDF, DOC, DOCX, XLS, XLSX, TXT, RTF support is built-in. You can add more file type support easily.

  • PNG, GIF, JPG, RAR, ZIP, PDF attachments support

  • Simple math human verification

  • Send email with or without attachments

Multiple Attachment Support

Firefox 3.6+, Safari 5+, Chrome 6+, Opera 11+, IE 10+

Above browsers or their higher versions will support multiple upload while versions prior to those will attach single file at a time but over all the script will run in every web browser.

Script Settings

Max Execution Time: Set maximum execution time for script.
Memory Limit: Set PHP memory limit according to your needs.
Max File Size: Max file size allowed is total size script should allow for attachments.
Multiple or Single Upload: Set $multipleUpload to yes to allow multiple upload and set $multipleUpload to no to disallow multiple uploads.

Allow / Disallow File Types (Old Method)

Common File Types

PNG image/png
GIF image/gif
JPG image/jpeg
RAR application/x-rar-compressed
ZIP application/zip
PDF application/pdf
XML text/xml
TXT text/plain
HTML text/html
MP3 audio/x-mp3

Consider this script below

            if(strstr($_FILES['ssFile']['type'][$i], 'image/png')!==false
                || strstr($_FILES['ssFile']['type'][$i], 'image/gif')!==false
                || strstr($_FILES['ssFile']['type'][$i], 'image/jpeg')!==false
                || strstr($_FILES['ssFile']['type'][$i], 'image/pjpeg')!==false
                || strstr($_FILES['ssFile']['type'][$i], 'application/x-rar-compressed')!==false
                || strstr($_FILES['ssFile']['type'][$i], 'application/zip')!==false
                || strstr($_FILES['ssFile']['type'][$i], 'application/pdf')!==false
              )

See after 180 lines in example.php

|| is a separator of condition. This whole part of the script checks whether the attached files have the file type (extension) like one of those mentioned in the conditions or not. If not, the script will stop and show error to the user that he is not allowed to attach certain file type.

Follow the pattern above to add more file type support in the script. Above mentioned file types are accepted in the system.

For example

|| strstr($_FILES['ssFile']['type'][$i], 'image/gif')!==false
|| strstr($_FILES['ssFile']['type'][$i], 'image/jpeg')!==false

Above code will allow GIF and JPG files to be attached.

Search for these lines of code in the script and remove or comment any of these lines to remove support for that file type.

Let's say you want to add support for XML file to be attached. You will have to add the following line in above code.

|| strstr($_FILES['ssFile']['type'][$i], 'text/xml')!==false

So the above code becomes:

|| strstr($_FILES['ssFile']['type'][$i], 'image/gif')!==false
|| strstr($_FILES['ssFile']['type'][$i], 'image/jpeg')!==false
|| strstr($_FILES['ssFile']['type'][$i], 'text/xml')!==false

Find the following line in the code:

$allowedFiles =  array('gif','png' ,'jpg' ,'jpeg' ,'rar' ,'zip' ,'pdf' ,'doc' ,'docx' ,'xls' ,'xlsx' ,'rtf', 'txt');

Changing Memory and File Size Limits for Script

PHP has default limits set for few things in php.ini file. PHP.ini file is not always available in all web hosting environments so you can adjust those limits by defining in the PHP scripts or via .htacces file.

PHP Memory Limit

PHP has memory limit for any script. You can increase / decrease / adjust it according to your needs.

Via PHP Script

@ini_set('memory_limit', "64M");

OR

Via .htaccess

php_value memory_limit 64M

File Size Limit for Upload and Post Data Through Form

PHP has got a limit for data size you can post through a form. And limit for data size that can be uploaded. Adjust it according to your needs by using code below.

Via PHP Script

@ini_set('post_max_size', "10M");
@ini_set('upload_max_filesize', "10M");

OR

Via .htaccess

php_value upload_max_filesize 10M
php_value post_max_size 10M

This will set allowed file size limit for 'upload' and 'post data' to 10 MB. These two are best to set via .htaccess.
